# Western Union React Client Project

This is a sample enterprise React frontend project designed around Western Union-style business flows such as money transfer tracking, customer KYC verification, transaction monitoring, payment status, agent operations, fraud review, and reporting dashboards.

## Main Features

- Transaction dashboard
- Customer profile and KYC status view
- Money transfer tracking
- Payment processing service layer
- Role-based navigation structure
- Redux Toolkit state management
- React Router page routing
- Reusable API service layer
- Enterprise-style folder structure

## Tech Stack

- React
- Vite
- React Router
- Redux Toolkit
- Axios
- JavaScript / JSX

## Run Locally

```bash
npm install
npm run dev
```
